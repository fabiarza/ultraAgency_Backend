 Backend del proyecto **UltraAgency**, desarrollado en ** Java 17**, con  **Spring Boot** y **Maven**.

 ---
 ## Descripción
 Este proyecto implementa la lógica del servidor de UltraAgency, incluyendo la gestión de datos,endpoints de la API y conexión a la base de datos.
 Actualmente es la versión inicial y se irá ampliando con nuevas funcionalidades.
 
 ---
 
  ## Tecnologías
   - Java 17
   - Spring Boot
   - Maven
   - Git / GitHub
   
  ## Instalación y ejecución

  ### Requisitos  
  - Java 17
  - Maven
  - IDE recomendado: IntelliJ IDEA, Eclipse o VS Code
  
  ### Pasos
    1. **Clona el repositorio:**
     ```bash
      git clone https://github.com/fabiarza/ultraagency-backend.git
     ```
   
   2. Accede a la carpeta:
        cd ultraagency-backend
   
   3. Compila y ejecuta:
        mvn clean install
	    mvn spring-boot:run
	 
   4. La aplicación se levantará en
     	 http://localhost:8080
		 
	Estructura del proyecto
	   ultraagency-backend/
	   │
       ├─ src/main/java/...       # Código fuente Java
       ├─ src/main/resources/...  # Archivos de configuración y recursos
       ├─ src/test/java/...       # Tests unitarios
       ├─ pom.xml                 # Configuración de Maven
       └─ README.md
